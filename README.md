# Sistema de Controle de Atendimento (Web Ticket)

Sistema simples de gerenciamento de filas desenvolvido em React (via CDN) para fins educacionais.

## Funcionalidades
- Emissão de senhas (Geral, Prioritário, Exames).
- Painel de chamadas em tempo real.
- Mesa do atendente para chamar e finalizar senhas.

## Como usar
Este projeto é um arquivo único (`index.html`). Para rodar:
1. Baixe o arquivo `index.html`.
2. Abra diretamente em seu navegador.
